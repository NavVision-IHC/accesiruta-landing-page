# AccesiRuta Landing Page

## Descripción

AccesiRuta es una solución digital desarrollada por la startup NavVision. Su objetivo es mejorar la movilidad urbana de personas con discapacidad motriz y visual mediante rutas accesibles, alertas en tiempo real, reportes colaborativos y asistencia por voz.

## Segmentos objetivo

### Personas con discapacidad motriz

Usuarios que necesitan rutas accesibles con rampas, veredas amplias, cruces seguros y menos obstáculos urbanos.

### Personas con discapacidad visual

Usuarios que necesitan guía por voz, alertas auditivas y orientación clara para desplazarse con mayor autonomía.

## Características principales

- Rutas accesibles.
- Reportes colaborativos.
- Alertas en tiempo real.
- Guía por voz.
- Filtros de accesibilidad.
- Interfaz clara e inclusiva.
